Catch-Score verification index

A plain-language guide to the typed checks shown on a record page.

What this index is

The index reports which checks were performed against a record and its source: source record, quote fidelity, provenance, primary verification, review, correction watch, correction history, and source alignment.

SAI is the evidence-governance system that records which checks ran against which sources, with typed gaps when a check has not run. SAI is not a truth-scoring system; the index preserves those check results without turning them into a truth judgment.

What this index is not

It is not a truth score of the underlying claim. It does not estimate a probability, rank records, or combine different checks into one number.
